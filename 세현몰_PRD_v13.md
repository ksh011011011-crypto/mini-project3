# 세현몰 제품 요구사항 문서 (PRD) v13

## 1. 개요

| 항목 | 내용 |
|------|------|
| 제품명 | 세현몰 (Sehyeon Mall) |
| 버전 | PRD v13 |
| 목적 | 소규모 브랜드 상품 온라인 판매 및 구매 전 1:1 화상 상담 연동 |

## 2. 목표 사용자

- 브랜드 상품을 온라인으로 구매하려는 일반 소비자
- 사이즈·옵션·배송 문의가 필요해 화상 상담을 원하는 사용자

## 3. 핵심 기능 범위 (MVP)

### 3.1 상품

- 상품 목록 조회, 카테고리 필터(간단)
- **빠른 메뉴**: 베스트(큐레이션)·이벤트(배너)·주문배송(장바구니 구역) 스크롤 이동
- **매장 층 안내(B1~12층 복합몰)**: **B1 세현마트**, 1층 패션·의류, 2층 리빙·홈, 3층 액세서리·잡화, 4층 뷰티, 5층 스포츠, 6층 키즈, 7층 디지털·가전, 8층 도서·문구, 9층 식품·푸드코트(입점별 메뉴 묶음), **10층 영화관·매점**, 11층 문화·공연·갤러리, 12층 스카이 라운지·티룸. 「전체 매장」에서는 층별로 구획되어 표시된다.
- **단지 시설(데모)**: 상단 「시설」줄에서 지하 주차장 안내, 9F 식당가·B1 마트·12F 스카이로 몰 층 이동, `?kiosk=1` 영화 키오스크(좌석 예매 UI), 「세현 시네마」탭으로 일반 영화관 홈
- 상품 상세: 이름, 가격, 설명, 이미지(원격 이미지 URL 허용, 로드 실패 시 대체 표시)
- 목록 카드에서 「자세히」로 상세 영역을 연다.

### 3.2 장바구니

- 상품 담기, 수량 변경, 로컬(브라우저) 기준 유지

### 3.2.1 가짜 결제·취소(데모)

- 관리자 계정(`ksh011` / `lms990302!!`) 로그인 후에만 주문·예매 결제가 완료 처리된다.
- 결제 모달에서 **취소** 시 장바구니·예매 데이터는 유지된다.
- 승인 처리 중 **「취소 · 승인 중단」**으로 타이머를 끊고 폼으로 돌아갈 수 있다.

### 3.3 Zoom 화상 상담 — **48시간 유효 링크**

- **정책**: 상담용 Zoom(또는 동급 화상) 참여 링크는 **발급(생성) 시점부터 48시간** 동안만 유효한 것으로 간주한다.
- **표시**: 남은 유효 시간을 사용자에게 노출한다(카운트다운 또는 만료 시각).
- **만료 후**: 링크는 비활성화 처리하고, 재발급 안내 문구를 표시한다.
- **비고**: 실제 Zoom SDK 연동 없이도, PRD 준수를 위해 **동일한 만료 규칙**을 적용한 데모 URL로 동작 검증 가능.

### 3.4 비기능

- 반응형 레이아웃(모바일 우선)
- 한국어 UI

## 4. 범위 외 (v13)

- 실결제(PG), 회원/로그인, 재고 연동, 관리자 백오피스
- Zoom 공식 API로 회의 생성(서버 사이드)

## 5. 성공 지표 (참고)

- 상품 탐색 → 장바구니 담기 플로우 완료
- 상담 링크 발급 후 48시간 이내/이후 UI 상태가 명세와 일치

## 6. 로컬 실행 (개발)

저장소 루트에서 `npm install` 후 `npm run dev` 로 프론트엔드 개발 서버를 실행한다.
