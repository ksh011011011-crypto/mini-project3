통합 웹 데모

몰 · 시네마 · 콘서트 · 주차를 한 화면에서 체험하고,
챗봇·BGM·전환 연출로 발표 임팩트를 높인 프론트엔드 시연용 앱입니다.

프로젝트 개요

복합 문화·상업 시설을 단일 포털에서 탐색하는 시연용 웹 애플리케이션입니다. 실제 결제·백엔드 없이 로컬 상태로 장바구니·예매·주차 흐름을 재현합니다.

시연 포인트

  • 탭(포털) 전환 시 시각 효과와 BGM 전환
  • 몰·시네마 화면의 별·반짝임 등 분위기 연출
  • 챗봇으로 각 시설 화면으로 빠르게 이동
  • 시네마: 가장 빠른 회차 + 추천 좌석 기준 자동 예매 데모

세현몰

쇼핑 플로우

카테고리·층(B1 등)·상품 탐색, 장바구니, 체크아웃 모달(승인 시뮬레이션·취소 가능).

연출

히어로 영역 별 레이어, 포털 전환·펄스와 연동된 FX.

시네마

예매 UX

영화·회차·좌석 선택, addBooking으로 즉시 예약 반영.

자동 예매

가장 빠른 회차 + 추천 좌석 2석 우선으로 원클릭 데모.

비주얼

별 실루엣 레이어 등 몰과 톤을 맞춘 배경 연출.

콘서트 · 주차

콘서트

공연 정보·티켓 플로우, 포털 샤인·FX 펄스로 다른 탭과 동일한 완성도.

주차

입차·요금·키오스크 등 단지 내 이동 맥락을 보완.

챗봇 & 네비게이션

자연어/버튼형 입력으로 몰 · 시네마 · 콘서트 · 주차 화면으로 전환합니다. 발표 시 “챗봇에 말하고 바로 해당 탭으로 이동”하는 시나리오에 적합합니다.

기술 메모

로컬 브레인(localChatbotBrain)과 액션 타입(ChatNavigateAction)으로 화면 전환을 트리거합니다.

사운드 & 인터랙션

BGM 페이드 포털 버스트 클릭 스파크

  • 시설별 앰비언트 BGM, 전환 시 페이드아웃(fadeOutHtmlAudio).
  • 포털(탭) 전환마다 여러 종류의 버스트·마운트 애니메이션.
  • 문서 전역 클릭 시 스파크(필요 시 data-no-click-spark로 제외).
  • 접근성: prefers-reduced-motion 시 연출 완화.

기술 스택 & 실행

  • React 18 + TypeScript + Vite 6
  • 상태·예매·장바구니: 주로 클라이언트 로컬 상태 / localStorage 패턴
  • 개발: npm run dev (기본 포트는 프로젝트 설정에 따름, 예: 5188)
  • 배포 시: npm run build 후 정적 호스팅
